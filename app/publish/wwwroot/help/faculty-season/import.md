# Importing Data from the "Triton" System

AcaTime supports importing data from Excel files generated by the "Triton" system used at Taras Shevchenko National University of Kyiv to manage the academic process.

## Required Files

Two types of files are needed:

1. **Working (Operative) Curriculum** — Excel files with educational program data
2. **Hour Distribution Report** — Excel files with teacher workload data

Package these files into a ZIP archive and upload it when creating a new semester.

## Step 1 — Prepare the Archive

1. Collect all Excel curriculum files for the current semester
2. Collect all Excel hour-distribution files for the current semester
3. Create a folder (e.g. `Import_Autumn_2025`) and copy all files into it
4. Select all files (Ctrl+A) → right-click → **Send to** → **Compressed (ZIP) folder**
5. Name the archive (e.g. `Import_Autumn_2025.zip`)

## Step 2 — Import in AcaTime

1. Log in as a Faculty Administrator
2. Navigate to **General** → **Academic Semester**
3. Click **Add Semester**
4. Fill in the **Main Data** tab (period name, dates, max periods per day)
5. Switch to the **Import** tab
6. Under **Import from file** click **Choose File** and select the ZIP archive
7. Click **Save**

The import may take a few moments depending on archive size.
If any warnings occur during import, they are shown in a popup after completion.

## What Gets Imported

**From curriculum files:**
- Educational programs
- Study disciplines
- Groups and subgroups
- Lesson counts by subject type

**From hour-distribution files:**
- Teachers
- Workload distribution
- Links between teachers and educational programs/disciplines
