# Faculty Management

The **Faculties** section lets system administrators manage the list of faculties. It is available to System Administrators only.

## Capabilities

- View all faculties
- Add new faculties
- Rename existing faculties
- Delete faculties

## Adding a Faculty

1. Click **Add Faculty**
2. Enter the faculty name in the new row that appears in the table
3. Changes are saved automatically

## Editing a Faculty

1. Double-click the faculty name in the table
2. Edit the name
3. Press **Enter** or click outside the cell to save

## Deleting a Faculty

1. Find the row in the table
2. Click **Delete** in the Actions column
3. Confirm the deletion

## Constraints

- A faculty cannot be deleted if users or other data are linked to it
- Faculty names must be unique
- Faculty name cannot be empty
