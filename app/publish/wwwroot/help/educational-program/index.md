# Educational Programs

## Overview

**Educational Programs** define the curricula that students follow. They are the key structural element linking student groups and study disciplines.

## Access

Faculty Administrators only.

**Navigation:** Main menu → **Academic Process** → **Educational Programs**

## Viewing Programs

Select filters to display the list:
1. **Academic Semester** — required
2. **Year of Study** — optional

The table columns are:
- **Program Name** — full name of the educational program
- **Year of Study** — the cohort year
- **Actions** — delete

## Adding a Program

1. Select the semester and year of study
2. Click **Add Educational Program**
3. A new row appears with the name "New Program"
4. Click the name to edit it
5. Changes save automatically

## Editing a Program

1. Click the program name in the table
2. Edit the name
3. Press **Enter** or click outside to save

> Only the name can be changed. The year of study cannot be changed for an existing program to avoid breaking the academic structure.

## Deleting a Program

1. Click the delete (trash) icon in the Actions column
2. Confirm

> A program can only be deleted if no student groups or study disciplines are linked to it. Remove or reassign all related elements first.

## Relationships

| Related entity | Relationship |
|---------------|--------------|
| Academic Semester | Programs are created within a semester |
| Year of Study | Each program belongs to one cohort year |
| Student Groups | Groups are created under a program |
| Study Disciplines | Disciplines are linked to programs |
