# Reference Dictionaries

AcaTime uses several reference dictionaries to classify and categorise core system elements. These dictionaries allow flexible configuration of the academic process.

## Access

Only Faculty Administrators (role 2) can manage reference dictionaries.

**Navigation:** Main menu → **General**

---

## Classroom Types

Classifies classrooms by purpose and equipment.

**Used for:**
- Categorising rooms by specialisation (general, laboratory, etc.)
- Matching with subject types for automatic room selection during scheduling

### Managing Classroom Types

Main menu → **General** → **Classroom Types**

The table shows the **Name** of each type.

**Add:** Click **Add** → a new row "New classroom type" appears → click the name to edit → changes save automatically.

**Edit:** Click the name → edit → press **Enter** or click outside.

**Delete:** Click the trash icon → confirm.

> A classroom type can only be deleted if it is not in use.

---

## Years of Study

Used in the **Years of Study** (Courses) menu item to categorise student groups by study year.

**Used for:**
- Marking the study year of a group (1st–6th year for bachelor/master)
- Grouping and analysing data by year

### Managing Years of Study

Main menu → **General** → **Years of Study**

The table shows the **Name** (e.g. "1st year", "2nd year").

**Add / Edit / Delete** — same inline editing approach as Classroom Types.

> A year of study can only be deleted if it is not in use.

---

## Subject Types

Classifies subjects by the type of class delivery.

**Used for:**
- Defining the nature of a class (lecture, practical, laboratory, etc.)
- Linking to classroom types for automatic room matching during scheduling

### Managing Subject Types

Main menu → **General** → **Subject Types**

| Column | Description |
|--------|-------------|
| Short name | Abbreviation (e.g. "lec", "prac", "lab") |
| Name | Full name |
| Classroom type | Associated classroom type |

**Add:** Click **Add** → fill in the dialog (short name, full name, classroom type) → **Save**.

**Edit:** Click the edit (pencil) icon → modify → **Save**.

**Delete:** Click the trash icon → confirm.

> A subject type can only be deleted if it is not in use.

---

## Subgroup Types

Available in **General** → **Subgroup Types**. Classifies how student groups are split.

**Used for:**
- Dividing students into subgroups for practicals and labs
- Organising classes for specific student categories (e.g. different foreign languages)

### Managing Subgroup Types

The table shows **Name** and **ID** for each type.

**Add / Edit / Delete** — same inline editing approach as Classroom Types.

> A subgroup type can only be deleted if it is not in use.

---

## How Dictionaries Are Used

| Dictionary | Used in |
|------------|---------|
| Classroom Types | Creating classrooms; scheduling rules |
| Years of Study | Creating groups; educational programs |
| Subject Types | Creating subjects; linking subjects to rooms |
| Subgroup Types | Creating subgroups; teacher workload |
