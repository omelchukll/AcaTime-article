# Group Subjects

The **Group Subjects** section lets you manage the subjects assigned to specific student groups, assign teachers, and configure subgroup splits.

## Access

Faculty Administrator (role 2) or Operator (role 3).

**Navigation:** Main menu → **Academic Process** → **Group Subjects**

## Filters

At the top of the page select:
- **Academic Semester**
- **Year of Study**
- **Student Group**

After selecting a group, the subject list appears along with the **Auto-assign Teachers** button.

## Subject List Columns

- Subject name
- Lesson count
- Student count
- Classroom type (if required)
- Assigned teacher
- Groups / subgroups

## Features

### Viewing Subjects

After selecting a semester, year, and group, the system shows two categories:
- **Unsaved subjects** — available for the group but not yet configured
- **Saved subjects** — already configured for this group

### Adding a Subject to a Group

1. Select the semester, year, and group
2. Find the subject in the unsaved list
3. Click **Save** or **Choose Teacher and Save**

### Assigning a Teacher

1. Select a subject
2. Click **Choose Teacher and Save**
3. Select a teacher from the dialog
4. Click **Select**

### Auto-assigning Teachers

1. Select the semester, year, and group
2. Click **Auto-assign Teachers**
3. The system assigns teachers based on workload and availability

### Splitting into Subgroups

1. Select an unsaved subject
2. Click **Split into Subgroups**
3. Choose the distribution type in the dialog
4. Click **Select**

> Subgroups must already be defined in the **Student Groups** section before using this feature.

### Merging Classes (Streaming)

1. Select a subject
2. Click **Merge Class**
3. Select an existing class from another group
4. Click **Select**

### Removing a Subject

**From a group:**
1. Select a saved subject
2. Click **Remove from Group** → confirm

**From a subgroup only:**
1. Select a saved subject with a subgroup
2. Click **Remove from Subgroup** → confirm

## Classroom Handling

The system automatically determines the required classroom type from the subject settings:
- If a type is specified it is shown on the subject card
- If no type is determined a warning is displayed
- If no classroom is needed an appropriate label is shown

## Recommendations

1. Create all disciplines and subjects before assigning them to groups
2. Use filters to quickly locate the target group
3. Assign teachers to all subjects before generating the schedule
4. Use auto-assign after manually assigning primary subjects
