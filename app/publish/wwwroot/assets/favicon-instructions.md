# Інструкції для заміни favicon.ico

Для того, щоб замінити favicon.ico у вашому проекті AcaTime, виконайте наступні кроки:

## 1. Створіть PNG версії favicon

1. Відкрийте файл `public/assets/favicon-png.html` у вашому браузері
2. Натисніть кнопку "Згенерувати PNG"
3. Клікніть правою кнопкою миші на кожному зображенні та виберіть "Зберегти зображення як..."
4. Збережіть файли як "favicon16.png", "favicon32.png" та "favicon48.png"

## 2. Конвертуйте PNG в ICO

1. Відвідайте сайт [ICO Converter](https://www.icoconverter.com/) або [ConvertICO](https://convertico.com/)
2. Завантажте всі PNG файли, які ви створили на попередньому кроці
3. Конвертуйте їх в один ICO файл, який містить всі розміри (16x16, 32x32, 48x48)
4. Збережіть отриманий ICO файл як "favicon.ico"

## 3. Замініть існуючий favicon.ico

1. Замініть файл `public/favicon.ico` новим файлом, який ви створили
2. Перезапустіть Angular додаток, щоб зміни вступили в силу

## Альтернативний спосіб (для досвідчених користувачів)

Якщо у вас встановлений Node.js, ви можете використати пакет `png-to-ico` для конвертації PNG в ICO:

```bash
# Встановіть пакет png-to-ico
npm install -g png-to-ico

# Конвертуйте PNG в ICO
png-to-ico favicon16.png favicon32.png favicon48.png > favicon.ico
```

## Примітка

Якщо ви не бачите зміни після заміни favicon.ico, спробуйте очистити кеш браузера або відкрити додаток в режимі інкогніто.
